# Store Listing Copy

## Product Name

Bros Selection Translator

## One-line Summary

Translate selected English text into Chinese or English dictionary definitions directly on the page.

## Short Description

Select English text on any page and open a lightweight in-page card with Chinese translation, phonetics, or English dictionary definitions.

## Long Description

Bros Selection Translator is a lightweight selected-text translation extension for Chromium browsers.

It is designed for reading workflows where users need fast inline meaning lookup without switching tabs.

Key capabilities:

- English to Chinese translation with selectable providers
- English word lookup with phonetic symbols and multiple Chinese senses when provider data is available
- English to English dictionary mode for single-word definitions
- Floating in-page translation card near the current text selection
- Toolbar popup for quick provider switching
- Full settings page with synchronized preferences

Privacy and permissions:

- the extension reads only the text actively selected by the user
- selected text is sent only to the configured online translation or dictionary provider
- settings are stored with browser sync storage
- no built-in analytics or account tracking is included
