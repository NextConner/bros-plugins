# 隐私政策

`Bros Selection Translator` 只会处理用户在网页中主动选中的文本。

## 扩展会做什么

- 在用户划选英文内容后读取当前选中文本
- 将选中的文本发送给用户当前配置的在线翻译或词典服务
- 将扩展设置保存到浏览器同步存储中

## 可能发送到第三方服务的数据

根据用户选择的来源，扩展可能会把当前选中文本发送到：

- `https://translate.googleapis.com/`
- `https://api.mymemory.translated.net/`
- `https://api.dictionaryapi.dev/`
- `https://api.datamuse.com/`

扩展不会上传整个网页内容，只会发送用户当前选中的文本。

## 本地存储的数据

扩展会通过 `chrome.storage.sync` 保存以下配置：

- 默认翻译方向
- 英译中来源选择
- 英英词典来源选择
- 是否自动翻译

## 数据收集说明

- 不包含账号系统
- 不包含统计或追踪 SDK
- 不会建立用户画像
- 本项目本身不会上传翻译历史

## 用户控制

用户可以控制：

- 是否启用该扩展
- 选中什么内容进行查询
- 使用哪个翻译/词典来源
- 是否开启自动翻译

用户可以随时卸载扩展，并通过浏览器设置清除扩展存储的数据。

## 发布说明

在发布到 Chrome Web Store 或 Edge Add-ons 前，请将本隐私政策托管到公开的 HTTPS 地址，并将该地址填写到商店页面中。
